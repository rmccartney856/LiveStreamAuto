WINDOWS:
If you are using the 32 bit version of OBS Studio use the 32 bit version of this plugin, if you are using 64 bit OBS Studio use the 64 bit one. (or both if you want)
Copy the advanced-scene-switcher.dll in the respective OBS Studio plugin folder. 
It is usually located at "C:\Program Files (x86)\obs-studio\obs-plugins\64bit" or "C:\Program Files (x86)\obs-studio\obs-plugins\32bit".
Rember to install the Visual C++ Redistributable for Visual Studio 2017, if you have not done so already. (See plugin overview page)

MAC OS:
Rigth click the OBS app inside your Applications folder and choose "Show Package Contents". 
Copy the advanced-scene-switcher.so file to Contents/Resources/obs-plugins.

Linux:
Copy the advanced-scene-switcher.so file into the OBS Studio plugin folder. 
The plugin folder is usually located under "/usr/lib/obs-plugins/"
